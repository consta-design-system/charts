# DualAxes

<RemadeInfo
  commponentName="DualAxes"
  apiUrl="https://g2plot.antv.vision/en/docs/api/plots/dual-axes"
/>

Этот компонент подойдёт, если нужно вместе проанализировать разные ряды или типы данных. Например, валюту и проценты, стоимость и количество — обычно, в зависимости от времени.

В этом случае значения разных показателей выводятся на одной оси Х с двумя разными осями Y.

- [Две линии](#две-линии)
- [Две линии со стилизацией](#две-линии-со-стилизацией)
- [Линия и столбцы](#линия-и-столбцы)
- [Масштабирование](#масштабирование)
- [Список свойств](#свойства)

Создайте объект с данными для графика и передайте в свойстве `data`. Объект представляет собой двумерный массив в формате:

`[объект с данными для левой оси Y, объект с данными для правой оси Y]`

## Две линии

```tsx
export const data = [
  { year: '1991', value: 3, count: 10 },
  { year: '1992', value: 4, count: 4 },
  { year: '1993', value: 3.5, count: 5 },
  { year: '1994', value: 5, count: 5 },
  { year: '1995', value: 4.9, count: 4.9 },
  { year: '1996', value: 6, count: 35 },
  { year: '1997', value: 7, count: 7 },
  { year: '1998', value: 9, count: 1 },
  { year: '1999', value: 13, count: 20 },
]

export function DualAxesExampleDualLine() {
  return <DualAxes data={[data, data]} xField="year" yField={['value', 'count']} />
}
```

### Две линии со стилизацией

```tsx
export const data = [
  { year: '1991', value: 3, count: 10 },
  { year: '1992', value: 4, count: 4 },
  { year: '1993', value: 3.5, count: 5 },
  { year: '1994', value: 5, count: 5 },
  { year: '1995', value: 4.9, count: 4.9 },
  { year: '1996', value: 6, count: 35 },
  { year: '1997', value: 7, count: 7 },
  { year: '1998', value: 9, count: 1 },
  { year: '1999', value: 13, count: 20 },
]

export function DualAxesExampleDualLineWithStyle() {
  const vars = useThemeVars()
  return (
    <DualAxes
      data={[data, data]}
      xField="year"
      yField={['value', 'count']}
      geometryOptions={[
        {
          geometry: 'line',
          smooth: false,
          color: vars.color.primary['--color-bg-brand'],
          label: {
            formatter: datum => {
              return `${datum.value}`
            },
          },
          lineStyle: {
            lineWidth: 3,
            lineDash: [5, 5],
          },
        },
        {
          geometry: 'line',
          smooth: true,
          color: vars.color.primary['--color-typo-success'],
          lineStyle: {
            lineWidth: 4,
            opacity: 0.5,
          },
          label: {
            formatter: datum => {
              return `${datum.count}`
            },
          },
          point: {
            shape: 'circle',
            size: 4,
            style: {
              opacity: 0.5,
              stroke: vars.color.primary['--color-typo-success'],
              fill: vars.color.primary['--color-bg-default'],
            },
          },
        },
      ]}
    />
  )
}
```

## Линия и столбцы

const uvBillData = [
  { time: '2019-03', value: 350, type: 'uv' },
  { time: '2019-04', value: 900, type: 'uv' },
  { time: '2019-05', value: 300, type: 'uv' },
  { time: '2019-06', value: 450, type: 'uv' },
  { time: '2019-07', value: 470, type: 'uv' },
  { time: '2019-03', value: 220, type: 'bill' },
  { time: '2019-04', value: 300, type: 'bill' },
  { time: '2019-05', value: 250, type: 'bill' },
  { time: '2019-06', value: 220, type: 'bill' },
  { time: '2019-07', value: 362, type: 'bill' },
]

const transformData = [
  { time: '2019-03', count: 800 },
  { time: '2019-04', count: 600 },
  { time: '2019-05', count: 400 },
  { time: '2019-06', count: 380 },
  { time: '2019-07', count: 220 },
]

export function DualAxesExampleStackedColumnLine() {
  return (
    <DualAxes
      data={[uvBillData, transformData]}
      xField="time"
      yField={['value', 'count']}
      geometryOptions={[
        {
          geometry: 'column',
          isStack: true,
          seriesField: 'type',
        },
        {
          geometry: 'line',
        },
      ]}
    />
  )
}

### Масштабирование

Если график не помещается в контейнер по ширине, вы можете добавить к нему слайдер — элемент, который позволяет перемещаться между частями графика. Описать объект со слайдером можно в свойстве `slider`, стилизовать его — с помощью хука [useSlider](?path=/docs/hooks-useslider--playground).

```tsx
export function DualAxesExampleSlider() {
  return (
    <DualAxes
      style={{ marginBottom: 'var(--space-m)' }}
      data={[data, data]}
      xField="year"
      slider={{
        start: 0,
        end: 0.5,
      }}
      yField={['value', 'count']}
    />
  )
}
```

## Свойства

<RemadeTableInfo apiUrl="https://g2plot.antv.vision/en/docs/api/plots/dualaxes" />

| Свойство | Тип и значения | По умолчанию | Описание |
| -------- | -------------- | ------------ | -------- |
| `data`  | `Data[]`   | -            | Объект с информацией для графика. Двумерный массив в формате:  [объект с данными для левой оси Y, объект с данными для правой оси Y]  |
| `xField`  | `string`       | -            | Поле с данными для отображения на горизонтальной оси        |
| `yField`  | `string`       | -            |  Поле с данными для отображения на вертикальной оси      |
| [`slider`](#масштабирование)  | `object`       | -            |  Объект с форматом данных для слайдера. Хук для стилизации — [useSlider](?path=/docs/hooks-useslider--playground)       |
| `className`       | `string`       | -            | Дополнительный CSS-класс компонента         |
