<RemadeInfo commponentName="Bar" apiUrl="https://g2plot.antv.vision/en/docs/api/plots/bar" />

Столбчатая диаграмма. Показывает значения одного или нескольких параметров за разные периоды или в разных ситуациях — рисует «столбики».

### Пример

```tsx
type Item = {
  name: string
  month: string
  value: number
}

const data: Item[] = [
  {
    name: 'London',
    month: 'Jan.',
    value: 18.9,
  },

  // ...
  {
    name: 'London',
    month: 'Feb.',
    value: 28.8,
  },
]

export function BarExample() {
  return (
    <Bar
      style={{ marginBottom: 'var(--space-m)' }}
      data={data}
      xField="value"
      yField="month"
      seriesField="name"
    />
  )
}
```
