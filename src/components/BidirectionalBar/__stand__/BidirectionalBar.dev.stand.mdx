## Данные

Массив с объектами для сравнения передаются в параметре `data`.

```tsx
const data: Item[] = [
  {
    name: 'Петя',
    value1: 13.4,
    value2: 12.3,
  },
  ...{
    name: 'Саша',
    value1: 165.2,
    value2: 72.9,
  },
];
```

За поле, по которому будет производиться групировка значений для сравнения, отвечает свойство `xField`.

Поля по которым будут производиться сравнения определяются в свойстве `yField`.

```tsx
export const BidirectionalBarExample = () => {
  return (
    <BidirectionalBar
      data={data}
      xField="name"
      yField={['value1', 'value2']}
      style={{ height: 500, padding: 'auto' }}
    />
  );
};
```

## Ориентация

График имеет два вида ориентация: вертикальную и горизонтальную. Ориентация указывается через свойство `layout`.

```tsx
<BidirectionalBar data={data} xField="name" yField={['value1', 'value2']} layout="vertical" />
<BidirectionalBar data={data} xField="name" yField={['value1', 'value2']} layout="horizontal" />
```

| Свойство                | Тип и значения                              | По умолчанию | Описание                                |
| ----------------------- | ------------------------------------------- | ------------ | --------------------------------------- |
| [`data`](#данные)       | `ITEM[]`                                    | -            | Массив с информацией о данных сравнения |
| [`xField`](#данные)     | `string`                                    | -            | Поле с по которым данные сортируются    |
| [`yField`](#данные)     | `['string', 'string']`                      | -            | Поля по которым происходит сравнение    |
| [`layout`](#ориентация) | `vertical` <code>&#124;</code> `horizontal` | -            | Ориентация графика                      |
| `className`             | `string`                                    | -            | Дополнительный CSS-класс компонента     |
